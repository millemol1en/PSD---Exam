### Instructions
1. Compiling & Running C
   - `gcc -o machine machine.c`
   - `./machine ex11.out 8`
2. Compiling & Running Java
   - `javac Machine.java`
   - `java Machine ex11.out 8`
   - Be careful! Some files don't take any input. The example above takes an input of `8`.
3. Run the commands:
   - `open commands.sh`
   - Look at the commands!
